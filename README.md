# 🌐 Web Scraping Job Market Analyzer Dashboard

## 📌 Project Overview
This project is a real-time Job Market Analyzer Dashboard developed using Python, BeautifulSoup, Pandas, Matplotlib, and Streamlit. The system performs web scraping on online job listings and visualizes hiring trends through interactive dashboards and charts.

---

## 🚀 Features
- Real web scraping using BeautifulSoup
- Job data extraction from websites
- CSV data storage
- Interactive dashboard using Streamlit
- Hiring location analysis
- Company hiring trends
- Search functionality
- Pie chart visualization

---

## 🛠 Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib
- Streamlit

---

## 📂 Project Files
- `scraper.py` → Performs web scraping
- `jobs.csv` → Stores scraped data
- `app.py` → Dashboard application
- `requirements.txt` → Required libraries

---

## ▶️ How to Run

### Install libraries
pip install -r requirements.txt

### Run scraper
python scraper.py

### Launch dashboard
python -m streamlit run app.py
